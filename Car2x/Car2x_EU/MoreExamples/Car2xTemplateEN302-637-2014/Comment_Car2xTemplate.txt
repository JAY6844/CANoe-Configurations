CANoe.Car2x - Template Configuration (Car2x)
--------------------------------------------

**************************************************************************
*                                                                        *
* Note: This configuration is intended to be used as template for new    *
* configurations.                                                        *
*                                                                        *
* 1) Copy the complete folder Car2xTemplate to the desired directory.    *
*                                                                        *
* 2) Delete this Desktop.                                                *
*                                                                        *
**************************************************************************

Template configuration with a Car2x database and extended data structures 
for typical ETSI Car2x application messages to provide comfortable 
handling with these application messages.
