FlexRay Bitmask analysis
******************
Demonstrates how you can carry out a bit mask analysis for FlexRay with the Scope option.

The sample application uses a AUTOSAR database instead of a conventional DBC file. 
The data in the AUTOSAR database is used to interpret raw data like the DBC format.
In this example, the AUTOSAR data are also used for message and signal selection.

Configuration description
The CANoe features are organized on individual desktops for better management 
and configuration.
