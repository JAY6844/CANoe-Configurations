This CANoe configuration shows, how to control CANoe with other Programming Languages like VB.NET and C#.
Enclosed are two example projects on VB.NET and C# which use the COM-Server.

The VB.NET and C# examples show how to Open CANoe configuration, Start/Stop Measurement, Call CAPL Functions, Read/Write System Variables and Signals.
