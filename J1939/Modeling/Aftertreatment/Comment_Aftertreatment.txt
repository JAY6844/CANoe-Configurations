J1939 Aftertreatment System Demo with J1939 NodeLayer
********************************************************

Exhaust Aftertreatment System (EAS)

This configuration simulates an exhaust aftertreatment system for 
commercial vehicles with SAE-J1939 communication. 

The nodes work with the J1939 NodeLayer. Additionally the demo includes
a test based on the Test Feature Set.

Appropriated Files:

- J1939_NL.dll
- J1939TestServiceLibraryNL.dll
