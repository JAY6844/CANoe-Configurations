MOST Demo: Visual Basic Script
--------------------------------------------------------------------------------

1) CANoe mit der Konfiguration "CANoe.MOST - System Demo" starten.

2) Das Script "VisualBasic_Script.vbs" über die 32 Bit Eingabeaufforderung
   starten: "cscript VisualBasic_Script.vbs"
   Auf 64 Bit Betriebssystemen kann die 32 Bit Eingabeaufforderung über das
   Startmenü der CANoe Installation (Unterverzeichnis "Tools") gestartet werden.

3) Beobachten Sie die Auswirkungen des Scripts:
   Zunächst wird die Messung gestartet.
   Dann wird das Abspielen der CD mehrfach pausiert und wieder gestartet
   und die Lautstärke wird mehrfach verändert.
   Zuletzt wird das Abspielen der CD und dann die Messung gestoppt.
