MOST Demo: Visual Basic Script
--------------------------------------------------------------------------------

1) Start CANoe using the configuration "CANoe.MOST - System Demo".

2) Start the script "VisualBasic_Script.vbs" via the 32 bit console:
   "cscript VisualBasic_Script.vbs"
   On 64 bit operating systems the 32 bit console can be started via the
   startmenu of the CANoe installation (submenu "Tools").

3) Watch the script at work:
   First the measurement is started.
   Then the CD is paused and started again severel times and the volume
   is changed several times.
   Last the CD and the measurement are stopped.
