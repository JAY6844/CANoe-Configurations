
1. Open configuration for integration or create a new configuration
2. Copy the folder "ConfigurationPanel" to the directory of the configuration
3. Add the ConfigurationPanel/Panels/CANDisturbanceMainConfigPanel.vxp file to your configuration
4. Add all *.sysvar files included within ConfigurationPanel/SystemVariables to your configuration
5. Add the ConfigurationPanel/Nodes/CANDisturbanceConfigurationPanel.can to your configuration
6. Activate the CAN Disturbance Option within the Hardwware/Network Hardware dialog of CANoe
