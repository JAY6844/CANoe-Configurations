This configuration demonstrates a simple implementation of a class 1/2 tractor ECU.

Press the key switch to start bus communication. The 'd' button activates the
automatic transmission. Use the slider to set the speed of the tractor.
The 'Brakes'-button reduces the speed and enables the stop lights.

The 'Engagement'-switch engages the rear PTO output shaft. The speed of the
output shaft depends on the engine speed of the tractor.
 
 The experimental implement receives some parameter groups of the tractor ECU
 and displays the values.
